User Guide

Introduction

  1. Add your audio files via Drag&Drop to the table or via the "ADD FILES" button. Only compatible files will appear in the table. Compatible files are:
    • Audio files: wave (.wav)
    • Annotation formats*: AnnotJSON (*_annot.json), TextGrid, TextTable

    * These files have to be named as same as the audio files.

  2. Because the OH-Portal only supports audio files with one channel (mono) it will ask you in some cases to choose. Select the preferred channel. If you are not sure, you can select both.
  3. Each column represents a step of the transcription chain. Deselect the checkbox under the column's label if you want to skip this step. For further information move the mouse over the column's label.
  4. Click on the "VERIFY" button. Now you can select the preferred language and check the processing queue before it starts. If all is OK, click on "OK"
  5. Now click on the button "START PROCESSING".
  6. Each line represents a task and each step has it's own status. The status is represented as an icon:
    (no icon): pending
    processing
    error
    finished
    skipped

    For further information move the mouse over an icon.

  7. When a step has finished you can download the results. For that move the mouse over an icon and click on an icon of results table to download it. If you click on the eye icon you can see the result without download.
  8. If the "Manual Transcription" step is enabled you can open the result with OCTRA. Click on the blue pencil. After you finished editing click on the green "SEND TRANSCRIPTION" button on the bottom. OCTRA will be closed and the result will be saved.
  9. If the last two steps are enabled the result is will be processed again. Both finished you can download the results the same way as before or open it with the EMU-Webapp.

How to download many results at once

You can download all results of a line or of a column:

Downloading all results of a line:

  1. Select the lines you like to download (pressing CTRL on Windows or CMD on Mac + click on the lines).
  2. Do a right-click on one of the selected lines and click on "Download selected lines"
  3. Select the preferred annotation formats and click on "GET PACKAGE".
  4. Click on "DOWNLOAD RESULTS"

Downloading all results of a column:

  1. Click on a column's label.
  2. Select the preferred annotation formats and click on "GET PACKAGE".
  3. Click on "DOWNLOAD RESULTS".
